# Install-OpenCV

## Info
BASH script to install or update OpenCV latest release in Debian-based systems (Debian, Ubuntu, LinuxMint)

* http://opencv.org/

## HowTo
* `git clone  https://github.com/FelipMarti/Install-OpenCV.git`
* `cd Install-OpenCV`
* `chmod u+x OpenCV-install.sh`
* `./OpenCV-install.sh`
